The Avalara AvaTax4J Adapter provides local object-oriented access to the AutoAddress+TM and AvaTax STTM web services for Java-based applications.

The adapter was built using Apache Axis WSDL2Java to build the stubs which were then enhanced to support a custom Profile SOAP header element and a WS-Security UsernameToken SOAP header element. It uses Apache WSS4Jto build the security token.
